# Advance Data Structure (ADSA)

## Program Name
Merge Sort in C

## Description
This program sorts an array of integers using the Merge Sort algorithm.

## How to Run

1. Compile the program:
   ```bash
   gcc mergesort.py -o mergesort
   ```

2. Run the program:
   ```bash
   ./mergesort
   ```

## Sample Input

```
5
5 3 1 4 2
```

## Sample Output

```
Sorted array:
1 2 3 4 5
```
